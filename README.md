## En qué estoy enfocado

- **Arquitectura SaaS real**: módulos, permisos por rol, auditoría, multi-tenant cuando aplica y patrones para escalar sin reescribir todo.
- **APIs seguras**: autenticación sólida (JWT/sesiones), RBAC/ABAC, validación estricta, rate limiting, logs trazables y hardening básico bien hecho.
- **Performance y DX**: caching donde suma, queries optimizadas, builds limpios, CI mínimo, y una experiencia dev que no te castigue por tocar una línea.
- **UX empresarial**: UI consistente, jerarquía clara, estados (loading/empty/error) y diseño que se siente producto, no demo.
